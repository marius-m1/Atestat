# Atestat

## Cerințe

- **Docker Desktop** instalat

## Instalare și rulare

### 1. Descarcă proiectul

- Click pe butonul **Code** → **Download ZIP**
- Dezarhivează folderul

### 2. Navighează în folder

```
cd <calea_către_folder>
```

### 3. Rulează aplicația

```
docker compose up --build
```

### 4. Deschide în browser

```
http://localhost:7860
```
